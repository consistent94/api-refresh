About Historical Figures Explorer

This application demonstrates API data extraction and ETL (Extract, Transform, Load) processes using:

  • Extract: Wikipedia REST API for biographical data
  • Transform: Data cleaning and structuring with Python
  • Load: Flask web interface for data display

Perfect for learning Python, API integration, and data engineering fundamentals!
